# Sumukh Teja Vanamala | Portfolio

A high-performance, minimalist scrollytelling portfolio designed with a "Designer B&W" aesthetic. Built using React and Vite for speed, D3.js for interactive visualizations, and vanilla CSS for refined, precise styling.

## Key Features

- **Minimalist Aesthetics**: A clean, premium black and white design with smooth transitions and subtle micro-animations.
- **Scrollytelling Experience**: Content reveals and transitions smoothly as you navigate through the deck.
- **Interactive Map**: A custom Mapbox implementation with D3-driven annotations showing my current residence in Hyderabad.
- **Dynamic Navbar**: Responsive navigation that adapts its visual state (shrinking, changing colors, or hiding) based on the scroll position.
- **Resume-Ready**: The repository includes my latest resume in both [PDF](resume.pdf) and LaTeX formats.
- **Fully Responsive**: Optimized for everything from large desktops to mobile devices.

## Tech Stack

- **Framework**: [React](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Visualization**: [D3.js](https://d3js.org/)
- **Mapping**: [Mapbox GL JS](https://docs.mapbox.com/mapbox-gl-js/guides/)
- **Styling**: Vanilla CSS (CSS3)
